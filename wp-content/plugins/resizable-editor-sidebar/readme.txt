== Description ==
An intuitive solution to make the default WordPress Gutenberg sidebar resizable.

<h3>Key points</h3>
<ul>
<li>Functions straight out of the box</li>
<li>An easy to use drag and drop interface</li>
<li>Preferences are saved directly to your machine so your editors can work along side you with their own preference.</li>
</ul>


== Installation ==
<h4>PHP 5.6 or greater is recommended before installation</h4>

<ol>
<li>Navigate to the plugins menu.</li>
<li>Search for 'Resizable Editor Sidebar'</li>
<li>Click 'Install Now' followed by 'Activate'</li>
<li>You'll now be able to drag the sidebar to a width that is suitable for yourself.</li>
</ol>
